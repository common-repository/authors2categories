== Description == 
This plugin was developed because I have one blog with multiple authors who have
their own categories. The users' category access is administered by the Authors2Categories
in Tools panel of the admin.

Select the user. Click on Edit User (Tools>Authors2Categories) and select the categories he/she
are allowed to post to. Click on Save when you are done.

== Installation ==
1. Unzip this plug-in zip gile.
2. Upload the author2categories.php to the '/wp-content/plugins/' directory of your WordPress instance.
3. Activate the plugin through the 'Plugins' menu in WordPress.



== Frequently Asked Questions ==
* To uninstall: 1. Deactivate the plugin 2. If you are sure you do not wish to save the users' categories and you will not use the plugin, drop the table from the WordPress database your instance is using.
